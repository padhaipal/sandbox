# Donor-page learner photos

The donor page (`index.html`, just above "What PadhaiPal Does") shows three field
photos. Save the three images here with these exact filenames:

- `learner-1.jpg` — **left** slot (smaller). The phone-in-hand / app-screen photo.
- `learner-2.jpg` — **center** slot (larger, more rounded). The group-of-children photo.
- `learner-3.jpg` — **right** slot (smaller). The boy-in-red photo.

JPG or PNG both work (the page references `.jpg`). Roughly landscape for the
center image and portrait for the two side images looks best, but `object-cover`
will crop any aspect ratio to fit.
